# Memoriant Patent Platform — Licensing & Editions

## Open Source (MIT License)

The core platform is free and open source under the MIT license:

- Core library (memoriant-patent-core)
- All search providers (PatentsView, USPTO ODP, Semantic Scholar)
- All analysis modules (101, 102, 103, 112, formalities)
- Patent drafters (provisional, non-provisional, PCT)
- Pipeline orchestrator
- DOCX/PDF export
- Claude Code skills and agents
- Docker Compose deployment
- SQLite standalone mode

## Premium Features (Future)

The following features are planned for a commercial edition:

### Team & Enterprise
- Multi-user collaboration on patent projects
- Role-based access control (admin, attorney, inventor)
- Team dashboard with project assignments
- SSO/SAML integration
- Audit compliance reports

### Advanced Analytics
- Patent portfolio management (multiple patents, family tracking)
- Automated portfolio quality scoring
- Competitive landscape analysis
- Technology trend mapping
- Citation network visualization

### Professional Services Integration
- Patent attorney marketplace / referral network
- Automated office action response suggestions
- USPTO e-filing integration (direct submission)
- Docketing system integration (CPi, Foundation IP)

### Support
- Community support (GitHub Issues) — free
- Priority support with SLA — paid
- Onboarding assistance — paid
- Custom deployment consulting — paid

## Pricing Philosophy

- **Individual inventors:** Always free (core platform)
- **Small teams (2-5):** Free core + optional premium features
- **Enterprise (5+):** Commercial license for premium features + support SLA
- **Patent attorneys/firms:** Commercial license with volume pricing

## Revenue Model

1. **Premium feature subscriptions** — team/enterprise features
2. **Support contracts** — priority support with SLA
3. **Consulting** — custom deployment, integration, training
4. **Attorney referral network** — commission on referrals to partner attorneys
